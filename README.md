# mds24_diploma
Reinforcement Learning for Moscow Exchange

This repository implements a Deep Reinforcement Learning agent for automated trading on the Moscow Exchange (MOEX). Using Proximal Policy Optimization (PPO), the agent learns optimal trading strategies by interacting with a realistic market simulation environment. Features include multi-asset portfolio management, risk-aware decision making, and real-time market data integration.

The system processes historical OHLCV data, technical indicators, and market microstructure to train an AI model capable of making profitable trading decisions while managing drawdowns. Implementation includes backtesting framework, performance metrics visualization, and deployment-ready architecture for live trading integration.

Key technologies: Python, PyTorch, Stable-Baselines3, pandas, MOEX ISS API. Ideal for quantitative finance research, algorithmic trading development, and reinforcement learning applications in financial markets.
